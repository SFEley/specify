swap_out_require!()

We need this just so that the tests don‘t fail when we are running the tests outside of a real rails project. Otherwise, the tests would fail with a file not found error, since db/example_data.rb is no where to be found

 6:   def swap_out_require!
 7:     Kernel.module_eval do
 8:       
 9:       # Thanks, Jay Fields:
10:       # http://blog.jayfields.com/2006/12/ruby-alias-method-alternative.html
11:       require_method = instance_method(:require)
12: 
13:       define_method(:require) do |string|
14:         unless string == "/db/example_data.rb"
15:           require_method.bind(self).call(string)
16:         end
17:       end
18:     end
19:   end
